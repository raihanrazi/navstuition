This plugin enables you to have a side bar widget with slider (slideshow) of featured images from posts from a selected category.

== Description ==

This plugin enables you to have a side bar widget with slider (slideshow) of featured images from recent posts (custom post type), 
from a selected category (hierarchical taxonomy), from comma separated list of tags. If post has no featured image, plugin tries 
to extract any 1st image from that post content.  Customize post title and background color, background color of slider navigation panel. 
Color is selected from WordPress built-in color pallete. Manual input is available also. Select animation type: slide or fade. 
Setup slider speed and widget dimensions. You can have as many sliders as you wish with different categories. 
This is mainly useful if you have a dedicated category for photo gallery posts and would like to display recent post title with its featured photo.

== Installation ==

Installing procedure:
1. Deactivate plugin if you have the previous version installed. (It is important requirement for switching to this version from a previous one.)
2. Extract "edit-post-expire.zip" archive content to the "/wp-content/plugins/posts-slideshow-widget" directory.
3. Activate "Posts Slideshow Widget" plugin via 'Plugins' menu in WordPress admin menu. 
4. Go to the "Settings"-"Posts SlideShow Widget" menu item and setup global parameters for this plugin. Then go to 'Appearance=>Widgets', 
drag and drop 'Posts Slideshow' widget to one of slots availabe, make final tuning to widget options in place.
